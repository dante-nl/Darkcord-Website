Make Discord even darker

Darkcord is a theme for Powercord and GooseMod with lots more dark colors.

After you have used Darkcord, you probably think Discord's dark mode is light.

How to install?

Darkcord is very easy to install on both Powercord and GooseMod, and apart from changes by the clients, they both look the same.

Please select what client modification you use

Installation for Powercord

To install Darkcord, you have to open the Terminal on macOS and Linux, or the Command Prompt on Windows.

  • macOS: Press ⌘ + Space and type "Terminal" and press Enter
  • Linux: Press CTRL + ALT + T
  • Windows: Press Windows key + R, type "cmd" and press Enter

Then you have to copy the command using the button below and paste it in your Terminal or Command Prompt

Copy command

After you did that, press Enter, reload your Discord. Then you can enable it via a plugin like Theme Toggler to enable it, if it isn't enabled automatically.

Installation for GooseMod

To install Darkcord, you have to go to the Theme Store that you can access via Discord's Home (where DMs are) and clicking Themes.

When you did that, you can search for "Darkcord" and clicking "Install". And you're done!
